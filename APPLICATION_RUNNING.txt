Problem: Cannot access http://localhost:3000
Solution:
   - Check if backend server is running: Get-Process node
   - Check MongoDB is running: mongod should be active
   - Check port 3000 is not blocked
   - Restart server: npm start

Problem: "Cannot connect to MongoDB"
Solution:
   - Ensure MongoDB is installed
   - Start MongoDB service (mongod)
   - Verify connection string in .env
   - Check database name "AgriMate"

Problem: Frontend doesn't load
Solution:
   - Ensure index.html path is correct
   - Check frontend/index.html exists
   - Clear browser cache
   - Try different browser
   - Check browser console for errors

Problem: APIs return 404
Solution:
   - Check backend server is running
   - Verify API endpoint syntax
   - Check CORS is enabled
   - Review backend/server.js for routes

Problem: No data showing in frontend
Solution:
   - Verify MongoDB has data (check collections)
   - Check API endpoints returning data
   - Clear browser cache
   - Check browser console for errors
   - Verify API_URL in frontend is correct
