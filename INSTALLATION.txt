âš™ï¸ TROUBLESHOOTING
==================

Problem: "Port 3000 already in use"
Solution:
  lsof -i :3000
  kill -9 <PID>
  OR use different port: PORT=3001 npm start

Problem: "Cannot connect to MongoDB"
Solution:
  - Start MongoDB: mongod
  - Check MONGO_URL environment variable
  - Server will fallback to in-memory storage
  - Check console for error messages

Problem: "Frontend is blank"
Solution:
  - Check browser console (F12 â†’ Console tab)
  - Check for JavaScript errors
  - Try incognito mode
  - Clear browser cache

Problem: "Voice assistant not working"
Solution:
  - Use Chrome or Edge (better support)
  - Allow microphone permission when prompted
  - Use HTTPS or localhost only
  - Check browser console for errors

Problem: "CORS errors"
Solution:
  - CORS is already configured in server
  - Verify server is running
  - Check API URL is: http://localhost:3000/api/...
  - Try from http://localhost:8000
