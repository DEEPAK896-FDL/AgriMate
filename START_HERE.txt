Problem: Cannot open frontend
Solution:
   1. Check frontend/index.html exists
   2. Try right-click > Open with Browser
   3. Or use Python server method (Option 2)

Problem: "Cannot connect to API"
Solution:
   1. Backend should be running (terminal with "npm start")
   2. Check http://localhost:3000/api/health works
   3. Port 3000 should be available
   4. If port used, stop other apps using port 3000

Problem: "No data showing"
Solution:
   1. Backend must be running (npm start)
   2. MongoDB must be running (mongod)
   3. Database "AgriMate" should exist
   4. Check browser console (F12) for errors

Problem: Voice Assistant not working
Solution:
   1. Check browser supports Web Speech API (Chrome, Edge)
   2. Grant microphone permission when asked
   3. Try in English first
   4. Check browser microphone settings

Problem: Location not detected for weather
Solution:
   1. Grant geolocation permission
   2. Check browser location settings
   3. WiFi might help with accuracy
   4. Falls back to default location (Chennai) if not available
